六、jQuery插件
	6.zTree(菜单)插件
		6.1. 使用步骤：
			6.1.1. 下载所需的文件
				下载地址：http://www.zTree.me/
			6.1.2 引入文件
				<!--引入jQuery文件-->
					<script src="js/jquery-3.3.1.js"></script>
				<!--引入zTree插件文件-->
					<script src="js/zTree/js/jquery.ztree.all.js"></script>
				<!--引入zTree提供的样式文件-->
					<link rel="stylesheet" href="js/zTree/css/zTreeStyle/zTreeStyle.css" />
					
			6.1.3.查看demo文档，学习应用demo.html
			
		6.2 在HTML页面中，使用<UL>元素用于显示树菜单
			id属性： 用于JS 方便查找
			class属性： 是zTree插件提供
			
				<ul id="treeDemo" class="ztree"></ul>
				
		6.3 $.fn.zTree.init(obj,setting,nodes)
		 	含义
		 		zTree树菜单的初始化方法
		 		创建zTree必须使用此方法
		 	参数
		 		obj:是jQuery对象，在HTML页面中哪里显示树菜单
		 		setting:用于配置当前树菜单
		 		nodes:用于显示的树菜单内容	
				
				例：
				<ul id="treeDemo" class="ztree"></ul>
				
				var setting={};
				var nodes=[
					{
						name:"北京市",
						children:[
							{name:"海淀区"},
							{name:"东城区"},
							{name:"西城区"}
						]	
					},
					{name:"天津市"},
					{name:"上海市"}
				];
				
				
				$(function(){
					$.fn.zTree.init($("#treeDemo"),setting,nodes);
				})
				
		6.4 setting:配置当前树菜单
			格式：{key:value}
			选项：
				treeId:zTree的唯一标识，等于<ul>元素的id属性值
				treeObj:是<ul>元素的jQuery获取方式（jQuery对象），$("")
				            作用：方便操作
				callback:封装一系列的方法(还有其他方法具体请查看文档)
					beforeMouseDown()方法：决定onMouseDown事件是否被触发
					onMouseDown()方法: 完成鼠标按下后的处理函数
				data:
					simpleData:简单数据模式(Array数组)
						页面中的数据都是来源于数据库(数据结构)
						作用：用于设置nodes中的数据结构
						语法:
							data:{
								simpleData:{
									enable:true
								}
							}
					注意：设置simpleData.enable值为true,必须设置simpleData的其他参数
						idKey:  节点中的id属性
						pIdKey: 父节点中的id属性
						rootPId:根节点
